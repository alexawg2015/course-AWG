Task. Создать субтему на основе Batik. Переопределить чекбокс.

1. Содать папку sub_rnd_two. В ней создать sub_rnd_two.info c необходимым описанием.
2. Добавить в папку скриншот, лого, фавикон.
3. Создать файл template.php, в нём функцию sub_rnd_two_checkbox(), в которой
  переопределить вывод checkbox.
4. Скопировать с Batik файл style.css. Добавить стили для чекбокса.

Тесты.
1. Продемонстрировать, что включена тема sub_rnd_two.
2. Продемонстрировать внешний вид чекбокса на странице сайта.
3. Включить тему Batik.
4. Продемонстрировать внешний вид чекбокса на странице сайта.
Внешний вид чекбоксов в п.2 и п.4 должен отличатся, но при этом все остальные
елементы должны быть идентичны (кроме логотипа).

Деплоит:
 - push в репозиторий, pull на сервер.
 - включить тему и сделать её по умолчанию.

Необходимое время - 1 час (+30 мин придумать, как его изменить).
